# BarberOS V2

Sistema SaaS de agendamento para barbearia.

## Stack
- Frontend: HTML/CSS/JavaScript puro
- Backend: Supabase (PostgreSQL + Auth + Realtime)
- Hosting: GitHub Pages

## Funcionalidades
- Agendamento online de servicos
- Painel administrativo
- Gestao de barbeiros e servicos
- Promocoes e descontos
- Relatorios financeiros

## Deploy
Este projeto usa GitHub Pages. O deploy e automatico ao fazer push para a branch main.
